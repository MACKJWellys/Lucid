# Lucid Reactive Ambient V1

## What was wrong before

- The live path stayed too exposed, so the headphones often felt like a softened copy of the room.
- The ambient bed was a static two-partial drone, which read as a dull hum instead of a living texture.
- Transients mostly excited filtered resonances without enough tail or spatial bloom, so events did not linger musically.

## What changed

- The direct monitor path is now darker, more controlled, and dynamically reduced so it supports presence without sounding like obvious duplication.
- The bed is now a slow chordal pad that follows ambient energy and changes harmony over time instead of holding one static interval forever.
- Onsets still trigger tuned resonances, but those resonances now feed a stereo bloom tail with gentle bell-like sparkles and longer, more graceful decay.

## How the new scene behaves

- Layer A keeps a filtered live presence for safety and orientation.
- Layer B turns sustained environmental energy into a soft harmonic field with slow chord motion and width.
- Layer C turns transients into restrained resonant droplets and shimmer trails, with extra ducking when content becomes speech-like.

## Current limitations

- Speech handling is heuristic rather than a true classifier, so some voices may still excite the scene in lively spaces.
- The harmony is intentionally simple and fixed to one calm chord cycle for version 1.
- The tail is lightweight by design, so it prioritizes polish and latency over very dense reverb complexity.
